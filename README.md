# Tushar_25scs2040000614_IILM_GN
# 🚀 Features
- 🎯 Real-Time Face Detection - Detects human faces instantly using webcam input.
- 🧠 Face Recognition System - Matches detected faces with stored dataset images for identification.
- 📸 User Registration (Add Face) - Easily add new users by storing their face images in the system.
- 🗂️ Dataset-Based Identification - Uses image dataset where each file name acts as a unique user label.
- 📝 Automatic Attendance Marking - Records attendance automatically when a face is recognized.
- 📅 Date & Time Logging - Saves attendance with accurate date and time in CSV format.
- ⚡ Fast & Lightweight Execution - Runs efficiently without requiring heavy resources.
- 🔄 Reusable & Scalable Structure - Easy to expand with more users or features.
- 🧩 Simple Project Architecture - Beginner-friendly and easy to understand code structure.
- 🔐 Contactless Attendance System - Eliminates manual attendance and improves hygiene.

# 📁 Project Structure
- Face_recognition_attendance
    - add_face.py - Add new user face images to dataset
    - Face_recognisation.py - Main program for face detection & attendance
    - attendance.csv - Stores attendance records (Name, Date, Time)
- dataset - Contains images of registered users
    - Lavya_kumar.jpeg
    - Tushar.jpeg
    - Vimlesh_Kumar.jpeg

# 🔭 Vision
To develop a smart, secure, and fully automated attendance system using face recognition technology that eliminates manual processes and enhances accuracy, efficiency, and user experience across educational institutions and organizations.
